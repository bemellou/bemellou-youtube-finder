# bemellou-youtube-finder

> YouTube influencer discovery + qualification engine for [Bemellou](https://bemellou.com).

**Live:** [chubiez-youtuber-finder-production.up.railway.app](https://chubiez-youtuber-finder-production.up.railway.app)

---

## What it does

Finds YouTube creators whose audiences fit Bemellou's profile (Gen Z, mental-wellness-adjacent, consumer brand affinity). Qualifies them on engagement, audience demographics, and content adjacency.

Uses an engagement-scoring model originally built at Black Forest Supplements, where it drove $3M+ in influencer-led revenue. Adapted for the consumer wellness space.

## Why it exists

The cost of a wrong influencer placement is high. The cost of finding the right ones manually is also high. This bot collapses the discovery + qualification step from hours to minutes.

## Stack

- Python + YouTube Data API
- Custom engagement-scoring model
- Railway hosting

## Status

- **Active in production**
- Used to source partners for Bemellou's launch month influencer campaign
